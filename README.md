# Dual-Mode-Robotic-Arm-with-On-Device-VLA-Inference

Designing a robotic arm that switches between two control modes: direct teleoperation via a sensor glove, and autonomous manipulation driven by a Vision-Language-Action model running locally on an NVIDIA Jetson Orin Nano. The goal is to keep inference fully on-device, no cloud dependency, using model compression techniques to get OpenVLA running within the Orin's memory constraints. ROS2 and Stateflow manage the mode transitions between manual and autonomous operation. The system also incorporates a digital twin for simulation and validation before physical deployment.
